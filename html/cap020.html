    06 - Efeito Paralax nas Imagens

    Uma correção simples para o p, para isso vamos adicionar um padding-bottom para dar um espaço dos parágrafos.

    Para adicionar as imagens no fundo, primeiramente vamos colocar um id em section no HTML com um termo para que entenda que ali vai entrar uma imagem, por exemplo, img01 e para outra imagem img02.

    Para colocar e configurar ele nas CSS é simples, adicione com background-image com a url de onde se encontra a imagen. ainda configurando o background, bote o tamanho cover e altere a posição com position. Para fazer o efeito paralax, utilize background-attachment em fixed.

    Pode colocar uma sobra no bloco para dar um efeito de profundidade com box-shadow.

    transforme as caixar p dentro dos article id="img" em caixas mais transparente para ver o conteúdo da imagem de fundo, faça isso com background-color, coloque preto e ajuste a transparência.

    07 - Hospedando o Projeto Cordel

    Terminando o projeto, abra o explorer, e ache os aquivos do projeto. Feito isso, crie um novo repositório local com o nome de projeto-cordel salve ele e cole os arquivos copiados, feito isso, publique o repositório.

    Para transformar em página web, lembre que para isso o repositório tem que ser público. Vá no repositório e entre em configurações, clique em pages e coloque Branch que por padrão vai estar em none em main. Depois de um tempo irá aparecer um link na parte superio, clicando nele a página vai abrir.

    Pelo fato do GitHub ser um serviço gratuito, ele pode levar um tempo pra carregar os recursos, como a criação de páginas e hospedagem.

    O link pode ser compartilhado para qualquer um enquanto estiver público no GitHub.
