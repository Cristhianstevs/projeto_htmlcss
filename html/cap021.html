    Capítulo 21

    02 - Sua primeira tabela em HTML

    Para fazer uma tabela, vamos usar a tag table. Com essa tag podemos usar as seguintes tags:

    • tr - Table Row (Linha da tabela)
    • td - Table Data (Dados da tabela)

    A tag td vai dentro da linha tr.

    Podemos usar classes para definir estilos especificos para cada um deles.

    Uma forma simples de tirar os espaços entre os dados (td) use o estilo border-collapse: collapse; em table.
